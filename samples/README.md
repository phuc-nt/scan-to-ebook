# Samples — curated demo / review outputs

Public, intentionally-tracked pipeline outputs for download, review, and demo.
Everything else under `output/` and all `*.epub` are gitignored — this folder is
the one exception (see negation rules at the bottom of `../.gitignore`).

Keep this folder small and curated: a few representative artifacts, not a dumping
ground. Generated scans / full books stay local under `output/`.

## Contents

| File | What it is |
|------|------------|
| `aragong-q1-quarter-38pages.epub` | EPUB from a quarter run (38 of 152 pages) of *Tuyển Tập Aragông* |
| `aragong-q1-quarter-38pages.md` | Merged book Markdown (OCR text) behind that EPUB |
| `aragong-q1-quarter-context.md` | Context pre-pass output: detected 2-page spreads + 30 proper-name canonicalizations |
